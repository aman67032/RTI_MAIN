Overview of State RTI Rules

While the Right to Information Act, 2005 is a central legislation applicable across India, each state has formulated its own rules for implementation of the Act within their jurisdiction. These state-specific rules primarily deal with procedural aspects of RTI implementation.

State RTI Rules typically cover aspects like application fees, cost of information, format of application, and appeal procedures.

Diversity in State RTI Rules

There is significant diversity in the RTI rules formulated by different states, particularly with respect to:

  • Application fees and charges for obtaining information
  • Mode of payment accepted
  • Word limits for RTI applications
  • Format of applications and appeals
  • Procedures for the functioning of State Information Commissions

State Information Commissions

Each state has established a State Information Commission (SIC) to implement the RTI Act within its jurisdiction. These commissions handle appeals against decisions of Public Information Officers at the state level.
