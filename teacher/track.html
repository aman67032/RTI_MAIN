RTI Application Timeline

Understanding the typical timeline of an RTI application can help you track its progress effectively.

1

Application Submission

Application is submitted to the Public Information Officer (PIO) with the requisite fee.

Day 0
2

Acknowledgment

PIO provides an acknowledgment receipt with a unique registration number.

Within 7 days
3

Processing

PIO processes the request and collects the required information from relevant departments.

Day 7-25
4

Response

PIO provides the requested information or reasons for denial if applicable.

Within 30 days
5

First Appeal

If response is unsatisfactory or not received, applicant can file a first appeal.

Within 30 days of response deadline
6

Second Appeal

If first appeal is unsuccessful, applicant can file a second appeal with the Information Commission.

Within 90 days of first appeal decision
