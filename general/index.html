Frequently Asked Questions

What is RTI and who can file it?
RTI (Right to Information) is a fundamental right that allows any Indian citizen to request information from public authorities. Any citizen can file an RTI application.

What is the time limit for getting RTI response?
Public authorities must provide the requested information within 30 days of receiving the RTI application. For matters concerning life and liberty, the time limit is 48 hours.
Can I file RTI online?
Yes, you can file RTI applications online through the RTI Online Portal for central government departments and some state departments that have online facilities.
